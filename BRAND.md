# Visage Brand Guidelines

**Typography-first • Calm • Professional • Timeless**

---

## Logo

### Primary Wordmark

The Visage wordmark uses **Libre Baskerville** — an elegant serif that conveys sophistication without pretension.

```
Font: Libre Baskerville
Weight: 400 (Regular)
Tracking: 0.02em
Case: Title case
```

**Usage:**
- Website header
- Documentation
- Marketing materials

### Uppercase Variant

For administrative contexts or when more authority is needed.

```
Font: DM Sans
Weight: 500 (Medium)
Tracking: 0.15em
Case: UPPERCASE
```

**Usage:**
- Admin dashboards
- CLI headers
- System status pages

### Monogram

The letter **V** for small UI contexts.

**Usage:**
- Favicon
- App icons
- Loading states

## Typography

### Font Stack

```css
--font-display: 'Libre Baskerville', Georgia, serif;
--font-body: 'DM Sans', Inter, system-ui, sans-serif;
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
```

### Hierarchy

| Element | Font | Size | Weight |
|---------|------|------|--------|
| H1 | Display | 3rem (48px) | 400 |
| H2 | Display | 2.25rem (36px) | 400 |
| H3 | Display | 1.5rem (24px) | 400 |
| Body | Sans | 1rem (16px) | 400 |
| Small | Sans | 0.875rem (14px) | 400 |
| Caption | Sans | 0.75rem (12px) | 500 |
| Code | Mono | 0.875rem (14px) | 400 |

## Components

### Buttons

**Primary** — Warm accent, for main actions
```css
background: #d4a574;
color: #0a0a0b;
border-radius: 12px;
```

**Secondary** — Subtle, for secondary actions
```css
background: #1f1f23;
border: 1px solid #27272a;
color: #fafaf9;
```

**Ghost** — Minimal, for tertiary actions
```css
background: transparent;
color: #a8a8a3;
```

### Cards

```css
background: rgba(24, 24, 27, 0.8);
backdrop-filter: blur(24px);
border: 1px solid #27272a;
border-radius: 16px;
```

## What to Avoid

- ❌ Gradients (except subtle shadows)
- ❌ Neon or bright colors
- ❌ AI imagery or neural network symbols
- ❌ Face icons or camera symbols
- ❌ Futuristic or sci-fi aesthetics
- ❌ Playful or cartoonish elements
- ❌ Marketing hype language

## Voice & Tone

**Professional but warm.** Confident without being boastful.

- Clear and direct
- Technically accurate
- Helpful, not salesy
- Calm and reassuring

**Example copy:**
> "Professional headshots, generated by AI, trained on you."

Not:
> "Revolutionary AI-powered headshot technology!"
